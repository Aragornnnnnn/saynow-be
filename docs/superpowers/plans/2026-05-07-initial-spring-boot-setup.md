# Initial Spring Boot Setup Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create the initial Java 21 and Spring Boot 4 backend project structure for `saynow-be`.

**Architecture:** Start with a single Spring Boot application under `com.saynow`. Use Gradle Kotlin DSL, Spring MVC, Validation, Actuator, JPA, PostgreSQL, and Flyway. Keep secrets out of Git by loading Supabase connection settings from environment variables.

**Tech Stack:** Java 21, Spring Boot 4.0.6, Gradle Kotlin DSL, Gradle Wrapper 9.4.1, PostgreSQL, Flyway, JUnit 5.
